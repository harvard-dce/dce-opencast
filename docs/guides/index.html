Communication & Help

Mailing lists

Discussions take place in several mailing lists:

  • Community
  • Users
  • Developers
  • German Community
  • Spanish Community

IRC channel

For occasional quick questions we have an IRC channel on Freenode: #opencast.

Technical meeting

Our weekly meeting takes place in the BigBlueButton conferencing system (room 1, password: welcome) on Tuesday at 15:00 UTC. Feel free to join!

Adopters meeting

Every last Wednesday of each month adopters meet at 15:00 UTC in the e-meeting system digiMar. Feel free to join!

Development

Code

You can find the codebase of Opencast in the Git repository hosted at Bitbucket.

Pull requests

New features and bug fixes are only accepted over pull requests. There's a list of open pull requests where you also can find the official reviewer.
